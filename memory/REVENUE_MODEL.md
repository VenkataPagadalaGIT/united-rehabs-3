# United Rehabs - Revenue Model Strategy

## Document Purpose
Strategic revenue planning document for post-launch discussion.

---

## 🎯 Revenue Model Options (Ranked by Fit)

### 1. **Treatment Facility Listings** (Primary - Recommended)
**How it works:**
- Free basic listing for all verified facilities
- **Premium listing**: $299-999/month for enhanced visibility, photos, detailed profiles
- **Featured placement**: $500-2000/month for top positions in search results

**Why it fits:**
- Facilities have marketing budgets and need qualified leads
- Aligns incentives - you want accurate data, they want visibility
- Recurring revenue

**Implementation:**
```
Free Tier:        Name, address, phone, basic info
Premium Tier:     Photos, detailed programs, insurance accepted, response time badge
Featured Tier:    Top of search, homepage placement, "Verified Partner" badge
```

---

### 2. **Lead Generation / Call Tracking** (High Revenue)
**How it works:**
- Facilities pay per qualified lead ($50-200 per call/form submission)
- You route calls through tracking numbers
- Only charge for calls over 60-90 seconds (filters spam)

**Why it fits:**
- Addiction treatment has high customer lifetime value ($20K-100K+)
- Facilities happily pay for qualified leads
- You already have the traffic

**Caution:**
- Must disclose this clearly (already in Affiliate Disclosure ✅)
- Never let payment influence which facilities appear in search
- Consider ethics - don't optimize for "most paying" over "best fit"

---

### 3. **Data Licensing / API Access** (B2B)
**How it works:**
- Researchers, policy makers, healthcare orgs pay for data access
- Tiers: Basic API ($99/mo), Full dataset ($499/mo), Enterprise (custom)

**Why it fits:**
- You've done the hard work of aggregating/verifying global data
- Universities, NGOs, pharma companies need this data
- Passive income once API is built

**Example customers:**
- Academic researchers
- Insurance companies
- Government health departments
- Pharma/treatment companies

---

### 4. **Sponsored Content / Native Ads** (Secondary)
**How it works:**
- "Sponsored by [Facility Name]" articles
- Educational content underwritten by treatment centers
- Must be clearly labeled

**Rules:**
- Always label as "Sponsored"
- Editorial independence on statistics (never let sponsors influence data)
- Limit to 1-2 sponsored pieces per 10 articles

---

### 5. **Display Advertising** (Easy but Low Value)
**Why deprioritize:**
- Low CPMs for health content
- Ads can look spammy on trust-focused site
- Better to have clean design + premium facility listings

---

## 💰 Phased Approach

### **Phase 1: Launch (Month 1-3)**
- **Free for everyone** - build traffic and trust
- Add Google Analytics to understand which pages get traffic
- Add simple "Contact this facility" forms (no monetization yet)

### **Phase 2: Validate (Month 3-6)**
- Reach out to 10-20 facilities manually
- Offer **free enhanced listings** in exchange for feedback
- Test: "Would you pay $X for this visibility?"
- Track which facilities get the most inquiries

### **Phase 3: Monetize (Month 6+)**
- Launch **Premium Listings** ($299-499/month)
- Start with facilities that already get traffic from you
- Pitch: "You got 47 profile views last month. Premium = 3x visibility"

### **Phase 4: Scale (Year 2+)**
- Add lead generation (call tracking)
- Launch data API for B2B customers
- Consider facility verification/accreditation badges

## ⚠️ Ethical Guardrails (Critical)

1. **Never rank facilities by payment** - Search results must be objective
2. **Disclose all financial relationships** - Already in Affiliate Disclosure ✅
3. **Don't exploit vulnerability** - No aggressive popups, no "Call NOW or else" tactics
4. **Verify facilities before featuring** - A scam rehab paying you hurts users AND reputation
5. **Keep data independent** - Statistics pages should never be influenced by advertisers
